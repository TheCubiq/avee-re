Image

Displays album art, image from file or URL.

If no custom path is set it, album art will be displayed, if no album art exists generated art will be displayed (i.e. artist and title name with color background). Animated GIFs are supported and GIFs transparency by using the color key.

Properties

General element properties

See General Properties.

Custom Image

Image to be displayed.

Measure Animation Speed

Speed of animation, used for GIF images.

Keep Aspect Ratio And Crop To Fit

If set, don't stretch the image, crop to fill the desired dimension. This property overrides KeepAspectRatio.

Keep Aspect Ratio

If set, don't stretch the image, fit the desired dimensions.

Blurred Border

If image cannot be fitted into desired dimensions, fill empty areas with a blurred image.

Shape

Specify to display image in other shapes than a rectangle.

Mask Image

Image or Composition used to mask out areas of CustomImage.

Mask Mode

If MaskImage is used, specify what color components to interpret as masked out.

Color

Color of the image.

Color To

If MeasureColorBlend is used, the second color to blend Color to.

Measure Color Blend

Value to specify color blending between Color and ColorTo.

Color Key Enabled

If image has no transparency/alpha, enable to use a specific color key instead.

Auto Detect Color Key

If set, tries to detect color key by looking at image border colors.

ColorKey

Manually set the color used for transparency.

Transparency Strength

Boost the transparent parts of the image.

Opacity Strength

Boost the opaque parts of the image.

BlurEnabled

If set, blurs the entire image.

BlurDivider

Helps to increase the BlurRadius value.

BlurRadius

Amount of the blur.

Generated Album Art Hint Shift Hue

Shifts hue for default image. (used when CustomImage is not set)

Generated Album Art Hint No Text

Hide text on default image. (used when CustomImage is not set)

Generated Album Art Hint Force Gen

If set, force to show default image.

Generated Album Art Color

Color to use for default image. (used when CustomImage is not set)
