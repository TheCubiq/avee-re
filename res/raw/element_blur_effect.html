Blur Effect

Displays blurred version of input composition/image.

Usually "Show Unblurred Content" is set to overlay original unblurred input content on top of blurred one.

Properties

General element properties

See General Properties.

Blend Mode Content

Blend mode for displaying original content (before blurring). Used when ShowUnblurredContent or ShowUnblurredContentUnder is set.

Color

Tint of the blur.

Source Composition Index

Index of Composition for use as input.

Mask Image

Image used to mask out areas.

Show Unblurred Content

Show original content (before blurring) over the blur.

Show Unblurred Content Under

Show original content (before blurring) under the blur.

Blur Radius

Amount of blur.

Blur Multiplier

Boosts BlurRadius property.

1 Layer Scale

Adjusts scale of blurred content.

2 LayerScale

If set above 0, shows second layer of blurred content. Adjusts scale of blurred content.

3 Layer Scale

If set above 0, shows third layer of blurred content. Adjusts scale of blurred content.
