Rgb Split Effect

Offsets RGB channels of target composition/image.

Helps to create chromatic aberration effect. It's possible to split in any other colors than Red, Green, Blue.

Properties

General element properties

See General Properties.

Blend Mode Content

Blend mode for displaying original content (before blurring). Used when ShowUnblurredContent or ShowUnblurredContentUnder is set.

Target Image

Specify Image or Composition used as input.

Color

Tint of the effect.

Show Unblurred Content

Show original content (before blurring) over the blur.

Show Unblurred Content Under

Show original content (before blurring) under the blur.

SplitMultiplier

Boosts SplitAmount property.

Split Amount

Amount of the color offset split effect.

Split Color 0

First color component to offset.

Split Color 1

Second color component.

Note that the first and third component is actually offsetted, the second stays in place. If you want to split only two colors like red and cyan, set this color component to black.

Split Color 2

Third color component to offset.
