Motion Blur Effect

Blurs target composition/image content according to movement.

Blurs input according to Position changes and Scale changes of this element. It's also possible to use different motion input by setting "MotionSource".

Properties

General element properties

See General Properties.

Blend Mode Content

Blend mode for displaying original content (before blurring). Used when ShowUnblurredContent or ShowUnblurredContentUnder is set.

Color

Tint of the blur.

Target Image

Specify Image or Composition used as input.

Show Unblurred Content

Show original content (before blurring) over the blur.

Show Unblurred Content Under

Show original content (before blurring) under the blur.

Blur Amount Multiplier

Amount of the blur.

Motion Source

Specify what to interpret as motion.

EffectTransform - Use this element Position and Scale.
Manual - Use variables.

Manual
Pos Blur Amount

Controls position motion.

Scale Blur Amount

Controls scale/zoom motion.

Relative Motion Mode

Useful to disable when MotionSource is set to Manual, as variables commonly output absolutes values.
