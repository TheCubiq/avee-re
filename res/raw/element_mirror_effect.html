Mirror Effect

Mirrors region of input composition/image. 

Mirrors half horizontaly, half verticaly or quarter of image.

Properties

General element properties

See General Properties.

Target Image

Specify Image or Composition used as input.

Mirror Mode

Specify which sides to mirror. (e.g. horizontal or vertical)

FlipMirror

Flips sides, left or right side to mirror.

Color

Tint of the effect.
