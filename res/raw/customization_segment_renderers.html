Segment renderers

Bars - Rectangular bars.
Line - Connected line.
SharpBars - Sloped rectangular bars.
RoundBars - Rounded rectangular bars.

Properties

Bars
Color From

Color of the first bar.

Color To

Color of the last bar.

Fixed Height

If set to other value than 0, bar height will not change.

Bar Height Multiplier

Adjusts height of the bars.

Bar Width

Width of the bars.

Bar Width Affected By Shape

If set, the width of the bar ends will expand according to the ShapePath.

Mirror

If set, bars will expand in both directions.

Line
Color From

Color of the first point.

Color To

Color of the last point.

Fixed Height

If set to other value than 0, line height will not change.

Bar Height Multiplier

Adjusts height of the line.

Mirror

If set, the line will expand in both directions.

Flip Every Other

If set, will flip height of every other point on the line.
