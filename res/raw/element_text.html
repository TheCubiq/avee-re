Text

Displays static text and various variables.

It can be used to show track artist, title, time and other values.

Properties

Text

Text to display.

Text enclosed in "{}" will be recognized as a variable that will be replaced by it's according value.
List of varible names:

{TimeCurrent}
{TimeRemaining}
{TimeLength}
{Artist}
{ArtistOrTitle}
{Title}
{Album}
{Fps}
{FrameTime}
{ArtistAndTitle}

Color

Color of the text.

FontSize

Font size of the text.

TypeFace

Specify font to use.
